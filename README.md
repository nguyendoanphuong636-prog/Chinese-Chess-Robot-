# CCR3 - Chinese Chess Robot 🤖♟️

Repository scaffold for a Chinese Chess Robot project with modules for vision, control, chess engine, firmware, hardware, simulation, and tests.

## Project structure

```text
Chinese-Chess-Robot-/
├── docs/
├── hardware/
├── firmware/
├── software/
├── simulation/
├── datasets/
├── tests/
└── scripts/
```

## Quick start

```bash
chmod +x ./scripts/setup.sh ./scripts/run.sh
./scripts/setup.sh
./scripts/run.sh
```
